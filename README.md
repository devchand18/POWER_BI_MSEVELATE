# Youth Hostel Infrastructure Analysis Across Indian States – Power BI Dashboard

## Project Overview

This project presents an interactive Power BI dashboard that analyzes the distribution of youth hostels across Indian states and locations. The dashboard provides insights into hostel infrastructure, geographic distribution, and yearly trends in youth hostel development across India.

The objective of this project is to use data visualization techniques to understand patterns in youth hostel construction and identify key regions with strong tourism infrastructure.

---

## Objectives

* Analyze the distribution of youth hostels across Indian states
* Visualize hostel locations geographically across India
* Identify states with the highest youth hostel infrastructure
* Analyze yearly trends in youth hostel development

---

## Tools & Technologies Used

* **Power BI** – Data visualization and dashboard development
* **Microsoft Excel** – Data preprocessing and dataset management
* **GitHub** – Project documentation and version control

---

## Dashboard Features

### KPI Indicators

The dashboard includes key performance indicators such as:

* Total Youth Hostels Constructed: **1090**
* Total Locations Covered: **425**
* Average Hostels per Location: **2.56**

### State-wise Analysis

A bar chart visualizes the distribution of youth hostels constructed across different states. This helps identify states with the highest concentration of hostels.

### Geographic Visualization

The map visualization highlights youth hostel locations across India, providing spatial insights into hostel distribution and tourism hubs.

### Location Distribution

The dashboard shows the distribution of youth hostels across various locations, helping identify areas with higher tourism activity.

### Trend Analysis

A line chart displays the yearly trend of youth hostel locations across financial years, helping understand growth patterns in hostel infrastructure.

---

## Key Insights

* **Haryana** has the highest number of youth hostels constructed.
* Youth hostels are widely distributed across northern and northeastern regions of India.
* Certain locations such as **Agartala, Agra, and Aizawl** show higher hostel density.
* The yearly trend indicates relatively stable growth in youth hostel infrastructure.

---

## Project Outcome

This dashboard helps stakeholders understand the distribution and trends of youth hostel infrastructure in India. The insights generated can support tourism planning and infrastructure development strategies.
